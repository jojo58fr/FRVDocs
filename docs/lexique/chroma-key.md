## Définition
Le **chroma key** est une technique de post-production qui consiste à isoler un sujet filmé devant un fond uni (souvent vert ou bleu) pour le superposer à un arrière-plan numérique. En VTubing, cette méthode permet de détacher le visage ou les mains capturées de l'arrière-plan du studio pour créer un overlay stylisé.

## Usage
- Fond vert/bleu avec chroma key dans OBS, Streamlabs ou VTube Studio.  
- Permet d'ajouter des décorations dynamiques ou des scènes immersives.  
- Utilisé en collaboration pour placer l'avatar dans un décor commun.
